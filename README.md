# LinuxBash

**Cross-Platform Bash-like Tool for Windows, macOS, and Linux**

---

## Release v1.0.0

This release provides precompiled binaries for three major platforms:

| File                         | Platform       | Notes                           |
| ---------------------------- | -------------- | ------------------------------- |
| `LinuxBash-win-x64.zip`      | Windows 64-bit | Extract and run `LinuxBash.exe` |
| `LinuxBash-macos-x64.zip`    | macOS 64-bit   | Extract and run `LinuxBash`     |
| `LinuxBash-linux-x64.tar.gz` | Linux 64-bit   | Extract and run `LinuxBash`     |

---

## Installation

1. Download the appropriate archive for your OS.
2. Extract the archive to a folder.
3. Make the executable runnable (Linux/macOS only):

   ```bash
   chmod +x LinuxBash
   ```
5. Run the executable:

   ```bash
   ./LinuxBash   # Linux/macOS
   LinuxBash.exe # Windows
   ```

---

## Features

* Lightweight Bash-like shell experience.
* Cross-platform support with no installation required.
* Designed for scripting, automation, and learning Bash on any OS.
